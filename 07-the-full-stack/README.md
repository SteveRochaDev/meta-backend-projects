# 07 - The Full Stack

This is the seventh course in the Meta Back-End Developer Professional Certificate program.

The goal of this course is to bring together multiple skills to build a **full-stack Django application**. You will set up development environments, refactor existing front-end and back-end components, and create new applications using your full-stack knowledge, combining **HTML, CSS, JavaScript, Django, and MySQL**.

By the end of the course, it is expected to be able to:  
- Explain common concepts related to full-stack development.  
- Use HTML, CSS, and JavaScript to develop well-structured, interactive, and responsive websites.  
- Build a full-stack application using Django that stores its data in models on a MySQL database and updates pages via forms and API endpoints.  
- Describe the different environments where web applications are deployed.

The final project focuses on **developing a full-stack Django app**, integrating front-end and back-end skills, implementing forms, API endpoints, and working with database models to create a functional and responsive web application.
