# 05 - Django Web Framework

This is the fifth course in the Meta Back-End Developer Professional Certificate program.

The goal of this course is to teach you how to **build, secure, and administer web applications** using the **Django web framework**. You will explore Django's core concepts, including models, views, templates, and the HTTP request-response cycle, while implementing best practices for web development and security.

By the end of the course, it is expected to be able to:  
- Design a Django web application using Python, HTML, and CSS.  
- Implement views, routes, and templates to handle HTTP requests and responses.  
- Create data models to define database tables and dynamic web forms.  
- Use the Django Template Language to build dynamic webpages that display database data.  
- Apply web security best practices and mitigate common security risks.  
- Test, share, and maintain a Django web application according to industry standards.

The final project focuses on building a **fully functional Django web application**, integrating database models, views, and templates, while ensuring proper security and usability. This hands-on experience consolidates skills in back-end web development using Django, preparing you for real-world web application projects.
