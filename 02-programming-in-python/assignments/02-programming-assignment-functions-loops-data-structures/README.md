# Programming Assignment: Functions, Loops and Data Structures

## Introduction
This lab focuses on creating a menu ordering system using Python. You will implement functions to calculate the subtotal, tax, and display a summary of the order. The lab introduces functions, loops, and data structures to store and process user input.

## Goal
The goal of this assignment is to practice creating functions, using for loops to iterate over data, and applying data structures to solve real-world problems. By completing this lab, you will create a menu ordering system that accurately calculates the total bill including tax.

## Objectives
- Create new functions to solve specific problems  
- Gain experience using for loops to iterate over different data collections  
- Create and use data structures to store, retrieve, and loop over data  

## Project Structure
- `ordering_system.py`  
- `README.md`  

## Instructions

1. Open the `ordering_system.py` file.
2. Run the script and, when prompted, enter three items from the menu (1 = espresso, 2 = coffee, etc.).
3. Implement the following functions:
   - `calculate_subtotal(order)` – accepts the order list and returns the sum of the item prices, rounded to 2 decimal places.
   - `calculate_tax(subtotal)` – calculates 15% tax of the subtotal and returns it rounded to 2 decimal places.
   - `summarize_order(order)` – returns a list of item names and the total (subtotal + tax), rounded to 2 decimal places.
4. Save your changes.

## Key Takeaways
- Functions allow modular, reusable code.  
- For loops are useful to iterate over lists and data structures.  
- Data structures such as dictionaries and lists help store and process complex data.  
- Rounding ensures monetary values are displayed correctly.
