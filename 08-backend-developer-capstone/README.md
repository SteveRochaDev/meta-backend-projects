# 08 - Back-End Developer Capstone

This is the eighth course in the Meta Back-End Developer Professional Certificate program.

The goal of this course is to consolidate your skills by completing a **capstone project**, demonstrating full-stack back-end development in a real-world scenario. You will compose a Django web application with multiple API endpoints, connect it to a MySQL database, implement a template system, and apply authentication and unit testing.

By the end of the course, it is expected to be able to:  
- Compose a back-end application using multiple skills.  
- Serve static HTML content using Django.  
- Commit the project to a Git repository.  
- Connect the backend to a MySQL database.  
- Implement menu and table booking APIs.  
- Set up user registration and authentication.  
- Test the application with unit tests and API tools like Insomnia.  
- Reflect on the project and synthesize skills from previous courses.

The final project focuses on creating a **portfolio-ready Django application**, showcasing your ability to integrate back-end and full-stack development skills, including APIs, databases, and testing.
