# Programming Assignment: Styling a Page

## Introduction
This lab focuses on styling an HTML page using CSS. You will practice applying different CSS rules including element, id, class, and descendant selectors.

## Goal
The goal of this assignment is to understand how CSS rules are applied to HTML elements and how selectors work. By completing this lab, you will style the content of an HTML page and preview the changes in a web browser using a local development server.

## Objectives
- Define a CSS rule using an element selector  
- Define a CSS rule using an id selector  
- Define a CSS rule using a class selector  
- Define a CSS rule using a descendant selector

## Project Structure
- `index.html`  
- `styles.css`  
- `logo.png`  
- `README.md`

## Instructions

1. Open the `styles.css` file.
2. Add a CSS rule for the `body` element that sets the background color to `#E0E0E2`.
3. Add a CSS rule for the `h1` element that sets the text color to `#721817`.
4. Add a CSS rule for the `h2` element that sets the text color to `#721817`.  
5. Add a CSS rule for the CSS class `center-text` that aligns the text to `center`.
6. Add a CSS rule for the element with id `logo`:
   - Set `margin-left` and `margin-right` to `auto`  
   - Set `display` to `block`
7. Add a CSS rule for all `span` elements that are children of `h2` elements:
   - Set text color to `#FA9F42`  
   - Set font size to `0.75em`
8. Add a CSS rule for the element with id `copyright`:
   - Set top padding to `12px`  
   - Set font size to `0.75em`

## Viewing Your Page
1. Start the live server by clicking **Go Live** in VS Code.
2. Note the exposed port number (e.g., `5500`).
3. Open the browser preview and navigate to:  
   `http://localhost:<port-number>`
4. Verify that the page displays the styled content correctly.
5. Stop the server after completing the lab.

## Key Takeaways
- CSS rules allow you to style HTML elements in many ways.  
- Selectors determine which elements a rule applies to.  
- Element, id, class, and descendant selectors are essential for targeting content.  
- Using a live server allows instant previewing of your styles.
