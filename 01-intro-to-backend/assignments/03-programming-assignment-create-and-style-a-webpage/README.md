# Programming Assignment: Create and Style a Webpage

## Introduction
This lab focuses on building a complete webpage using HTML and CSS. It combines structural HTML elements with basic styling techniques to create a simple personal webpage layout.

## Goal
The goal of this assignment is to create a structured HTML document and apply CSS styling rules to enhance its presentation. By completing this lab, you will demonstrate your understanding of HTML structure, lists, images, links, and CSS styling.

## Objectives
- Create a valid HTML5 document structure  
- Add an image to a webpage  
- Add headings, unordered and ordered lists  
- Add a hyperlink  
- Apply CSS styling rules to specific elements  
- Use element and ID selectors in CSS  

## Project Structure
- `index.html`
- `styles.css`
- `photo.jpg`
- `README.md`

## Instructions

### Part 1: Create the HTML File

1. Set up a valid HTML5 document structure.
2. Add your name inside the `<title>` element.
3. Link the `styles.css` file inside the `<head>` section.
4. Add five `<div>` elements inside the `<body>`.
5. In the first `<div>`, add an `<h1>` displaying your name.
6. In the second `<div>`, add `photo.jpg` using an `<img>` element and include `id="photo"`.
7. In the third `<div>`, add:
   - An `<h2>` titled **Favorite Music Artists**
   - An unordered list (`<ul>`) with five list items
8. In the fourth `<div>`, add:
   - An `<h2>` titled **Favorite Films**
   - An ordered list (`<ol>`) with five list items
9. In the fifth `<div>`, add a hyperlink to:
   - `https://www.meta.com/`
   - Use the link text: **My Profile**

### Part 2: Style the Webpage Using CSS

1. Add a rule for the image with `id="photo"`:
   - `border: 2px solid blue;`
2. Add a rule for `<h1>`:
   - `color: blue;`
3. Add a rule for all `<h2>` elements:
   - `color: grey;`
4. Add a rule for all `<div>` elements:
   - `margin: 4px;`

## Viewing Your Page

1. Save all files.
2. Click **Go Live** in VS Code.
3. Open the browser preview and navigate to:
   `http://localhost:<port-number>`
4. Confirm:
   - The image has a blue border
   - Your name is blue
   - All `<h2>` headings are grey
   - All sections have spacing (margin)
5. Stop the live server after completing the lab.

## Key Takeaways
- HTML provides the structure of a webpage.
- CSS enhances presentation and layout.
- Proper linking of external stylesheets is essential.
- Element and ID selectors are fundamental in CSS styling.
- Clean structure improves readability and maintainability.
