# 01 - Introduction to Back-End Development

This is the first course in the Meta Back-End Developer Professional Certificate program.

The goal of this course is to provide an introduction to back-end web development, featuring the daily responsibilities of back-end developers and the essential technologies that power server-side applications. 
This includes **Python**, **Linux commands**, **Git**, **SQL**, and an overview of **front-end technologies** like HTML, CSS, and modern UI frameworks for context.

By the end of the course, it is expected to be able to:  
- Distinguish between front-end, back-end, and full-stack developer roles.  
- Describe the day-to-day responsibilities of a back-end developer.  
- Explain the core and underlying technologies that power the web.  
- Use basic HTML and CSS to structure and style simple webpages.  
- Understand the benefits of using UI frameworks like Bootstrap and React.  

The final project focuses on creating and editing a **responsive webpage** using HTML and the Bootstrap CSS framework. Using a responsive layout grid, you will construct a webpage containing text and images that looks great on any screen size.

Note: This course serves as the foundation for the Meta Back-End Developer Professional Certificate, preparing learners for subsequent programming, database, and server-side courses.
