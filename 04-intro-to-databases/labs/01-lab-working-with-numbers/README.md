# Working with Numbers in MySQL

## Description
This lab demonstrates how to create and work with numerical values in a MySQL database. We created a `cm_devices` database to store information about mobile devices, including device ID, device name, and price. Additionally, we practiced creating a `stock` table to store quantity and total price information for devices in stock.

## Tasks Completed
- Created the database `cm_devices`
- Created the table `devices` with the following columns:
  - `deviceID` INT
  - `deviceName` VARCHAR(50)
  - `price` DECIMAL
- Verified table creation with `SHOW TABLES;` and `SHOW COLUMNS FROM devices;`
- Completed additional task by creating the `stock` table:
  - `deviceID` INT
  - `quantity` INT
  - `totalPrice` DECIMAL
- Tested all tables and verified their structures in MySQL
