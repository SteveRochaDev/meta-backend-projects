# Lab 7 – Record Deletion

## Description
This lab demonstrates how to delete records from a database table using the SQL `DELETE` statement.

## Tasks Completed
- Used existing `bookshop` database
- Cleared the `customers` table using `TRUNCATE`
- Inserted sample customer data
- Deleted a specific record using:
  - `DELETE FROM customers WHERE customerID = 3;`
- Verified results using `SELECT * FROM customers;`

## Optional Task
- Deleted another customer (Yasmine) using:
  - `DELETE FROM customers WHERE customerID = 6;`
