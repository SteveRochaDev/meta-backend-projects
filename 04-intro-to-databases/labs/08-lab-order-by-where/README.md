# Lab 8 – ORDER BY and WHERE

## Description
This lab demonstrates how to sort and filter data in SQL using the `ORDER BY` and `WHERE` clauses.

## Tasks Completed
- Created and used the `Chinook` database
- Created the `Customer` table with multiple attributes
- Inserted sample customer data
- Retrieved specific columns using `SELECT`
- Sorted results alphabetically using:
  - `ORDER BY FirstName`
- Filtered results using:
  - `WHERE Country = "Brazil"`
- Combined filtering and sorting in one query
