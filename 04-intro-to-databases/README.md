# 04 - Introduction to Databases for Back-End Development

This is the fourth course in the Meta Back-End Developer program.

The goal of this course is to provide an introduction to **databases**, including their core concepts, types, and management systems. You will learn how to create, manipulate, and query databases using **SQL**, and gain practical experience with relational database design and basic MySQL commands.

By the end of the course, it is expected to be able to:  
- Demonstrate a working knowledge of the concepts and principles behind databases.  
- Identify and explain the different types of database management systems.  
- Write and interpret basic SQL statements and commands.  
- Manipulate records in a database using SQL.  
- Plan and design a simple relational database system.  
- Understand alternatives to SQL and their use cases.

The final project focuses on applying these concepts to **design and interact with a simple relational database**, providing hands-on experience with MySQL and database management systems.
